# Shadow of the Five Winds
## Practical Solo-Dev Roadmap

---

# DEVELOPMENT PHILOSOPHY

Build the GAME first.
Build the WORLD second.
Build the PRESENTATION last.

A playable, fun first region is more valuable than:
- 40 unfinished maps
- cinematic cutscenes
- complicated lore systems
- ultra-polished menus

Your first goal is:
> "Can someone play this for 20 minutes and enjoy it?"

Everything else comes later.

- restart the project

# MOST IMPORTANT RULE

FINISHED > PERFECT

Players forgive:
- simple graphics
- small scope
- short runtime

Players do NOT forgive:
- abandoned projects
- boring gameplay
- unfinished games
